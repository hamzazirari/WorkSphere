WorkSphere – Gestion du Personnel
Description

WorkSphere est une application web interactive permettant de gérer les employés sur un plan d’étage en temps réel.
Elle permet d’ajouter, déplacer et supprimer des employés, tout en respectant les règles métier selon leur rôle et les zones autorisées.

Fonctionnalités

Ajouter un nouvel employé avec :

Nom, rôle, photo, email, téléphone

Expériences professionnelles multiples

Visualiser les employés dans une liste “Unassigned” et sur le plan d’étage.

Assigner un employé à une zone autorisée via le bouton “+”.

Supprimer un employé d’une zone avec le bouton “X”.

Voir le profil complet d’un employé en cliquant sur sa carte.

Respect des contraintes métier :

Réception → uniquement Réceptionnistes

Salle des serveurs → Techniciens IT

Salle de sécurité → Agents de sécurité

Manager → peut être partout

Nettoyage → partout sauf archives

Limitation du nombre d’employés par zone.

Interface responsive et moderne avec Tailwind CSS.

Technologies utilisées

HTML5 pour la structure

CSS3 et Tailwind CSS pour le design

JavaScript pour la logique interactive

Font Awesome pour les icônes

Installation / Utilisation

Cloner le repository :

git clone <lien-du-repository>


Ouvrir index.html dans votre navigateur.

Ajouter des employés via le bouton “Ajouter un nouvel employé”.

Cliquer sur le bouton “+” dans chaque zone pour assigner les employés.

Cliquer sur le bouton “X” pour retirer un employé d’une zone.

Cliquer sur un employé pour voir son profil détaillé.
